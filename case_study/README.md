# CS3 Case Study Package

This folder packages the KALz Project 3 deepfake image detection work for a second-year UVA student.

## Files

- `Angad_CS3.docx`: editable Word version of the case study packet.
- `Angad_CS3.pdf`: rendered print/check version of the same packet.
- `hook.md`: one-page hook document source.
- `student_rubric.md`: student-facing rubric source.
- `materials_and_references.md`: repo materials, source list, and hard-copy checklist.

The project code, model outputs, confusion matrices, Grad-CAM figures, and metric files live in the repository root under `scripts/` and `output/`.
